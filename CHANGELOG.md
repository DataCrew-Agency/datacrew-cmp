## [1.0.0] - 2024-XX-XX

### Added
- Initial release
- Google Consent Mode v2 support
- Customizable consent banner UI
- Cookie-based consent storage
- Multi-language text configuration
- GTM Consent Overview integration
- Granular consent categories:
  - Necessary (always on)
  - Analytics
  - Marketing
  - Functionality
- Accept All / Reject All / Save Preferences buttons
- Configurable banner position (top/bottom)
- Customizable colors and styling
- Region-specific default consent settings
